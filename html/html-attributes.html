HTML Attributes — विस्तृत मार्गदर्शिका

यह पेज HTML attribute क्या होते हैं, उनकी श्रेणियाँ, सामान्य attributes और उदाहरण (live demo) बताता है — सभी हिंदी में।

1. Attribute क्या हैं?

Attributes किसी HTML element को अतिरिक्त जानकारी देते हैं — जैसे element का पहचान (id), क्लास, लिंक की destination, input का आकार, आदि। सामान्य रूप से attribute name="value" के रूप में लिखा जाता है।

2. Attribute के प्रकार

  • Global attributes — सभी HTML elements पर लागू हो सकते हैं। (उदा. id, class, style, title, data-*)
  • Specific attributes — केवल कुछ tags पर लागू होते हैं। (उदा. href for <a>, src for <img>, alt, placeholder for <input>)
  • Boolean attributes — value की ज़रूरत नहीं होती; presence ही true बताती है। (उदा. disabled, readonly, checked, required)
  • ARIA attributes — accessibility के लिए (उदा. aria-hidden, aria-label)

5. Best practices

  • Semantics रखें — attribute का इस्तेमाल केवल तभी करें जब जरूरी हो।
  • Inline styles (style) कम उपयोग करें; CSS class बेहतर है।
  • alt और ARIA attributes का प्रयोग करें ताकि website accessible रहे।
  • data-* का प्रयोग custom data के लिए करें, ताकि markup semantic बना रहे।
  • Boolean attributes सिर्फ मौजूद रहें — disabled लिखें, disabled="disabled" भी काम करेगा पर अधिक शब्दों में न लिखें।
  • Form elements के लिए सही name और type रखें ताकि server-side पर data आसानी से मिले।

6. Common mistakes

  1. Attribute names में टाइपो (उदा. clas instead of class) — browser ignore कर देता है।
  2. Missing alt on images — accessibility और SEO दोनों प्रभावित होते हैं।
  3. Using inline style for complex layouts — maintainability खराब होती है।
  4. Relying on id for styling कई बार component reuse रोक देता है — prefer class when possible.
