HTML (HyperText Markup Language) is the backbone of any web page. An **HTML Element** defines a piece of content on the webpage. It is the most fundamental building block, essential for anyone starting their coding journey with X3School.

Anatomy of an HTML Element

An HTML element typically consists of three parts: an **Opening Tag**, the **Content**, and a **Closing Tag**.

Structure Example

<p>This is the content of the element.</p>
  • Opening Tag: <p> - Marks where the element begins.
  • Content: This is the content of the element. - The information displayed to the user.
  • Closing Tag: </p> - Includes a forward slash (/) and marks where the element ends.

Attributes

Attributes provide additional information about the element. They are always defined inside the **Opening Tag**, typically in name/value pairs (name="value").

Attribute Example (Using A Tag)

The <a> tag uses the href attribute to define the link destination.

<a href="https://x3school.in" target="_blank">Our Website Link</a>
  • Tag: <a> (Anchor Tag)
  • Attribute 1: href (Specifies the URL for the link)
  • Attribute 2: target="_blank" (Instructs the browser to open the link in a new tab)

Block vs. Inline Elements

HTML elements are broadly categorized into two display categories based on how they affect the layout of the document.

1. Block-Level Elements

  • They always start on a new line.
  • They take up the full width available, pushing elements after them down.
  • Examples: <div>, <h1> to <h6>, <p>, <section>, <form>.

2. Inline Elements

  • They do not start on a new line.
  • They only take up as much width as necessary for their content.
  • Examples: <span>, <a> (anchor), <img>, <em>, <strong>.

Code Example: Block and Inline

<!-- Block Elements -->
<div style="background: #FFF4A3;">First Div (Takes full width)</div>
<p style="background: #FFC0C7;">A Paragraph (Also takes full width)</p>

<!-- Inline Elements -->
<span style="background: lime;">First Span</span>
<a style="background: cyan;" href="#">An Inline Link</a>
<span style="background: lightblue;">Second Span</span>

Top HTML Element Interview Questions

  1. What is the difference between an HTML Element and an HTML Tag? Answer: Tags are the symbols (<p>, </p>) used to create the element, while the Element is the whole unit including the opening tag, content, and closing tag.
  2. What are Empty Elements (or Void Elements)? Give examples. Answer: Elements that do not have a closing tag and cannot contain content. Examples: <img>, <br>, <hr>.
  3. What is the main difference between <div> and <span>? Answer: <div> is a block-level element used for major structural groupings, and <span> is an inline element used for minor content groupings.
  4. What are Global Attributes? Answer: Attributes that can be applied to ALL HTML elements (e.g., class, id, style, title, data-*).
